Усложнением 6ой миссии серии является то, что теперь, для освежения помещения может быть нужно более одной лампочки. И это 5ый аргумент функции - сколько лампочек нужно для освежения помещения

Например, если для освещения помещения нужно 3 лампочки, то мы не защитываем то время, когда лампочек было всего 2 или только одна. Если последний аргумент функции не передан, значит достаточно одной лампочки для освежения помещения.

задача все таже - узнать сколько времени помещение было освещено (в этой задаче можно сказать - достаточно освещено)

Input: Two arguments. Both are ints.

Output: Int.
